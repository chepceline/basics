profoundgrid



A responsive grid system for fixed and fluid layouts.Built with SCSS, it gives you flexibility and full control.



Responsive

Make the grid fluid or fixed and add per-media layouts as needed. Change the column and gutter widths and remove or add columns as needed

Semantic

The Grid Keeps your markup semantic and works without any "grid-x" classes. Unless you want to use them, in which case they are included

Precise

Profound Grid uses negative margins to calculate columns. Unlike the other grid systems, fluid layout will look exactly the same in every browser
